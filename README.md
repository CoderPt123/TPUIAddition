## Installation

TPUIAddition is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "TPUIAddition", :git => "https://git.oschina.net/TPSky/TPUIAddition.git"
```
